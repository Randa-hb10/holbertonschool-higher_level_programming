# Python - Test Driven Development

This project is part of the Holberton School curriculum and focuses on
Test Driven Development (TDD) using Python.

## Learning Objectives

- Understand why Python is powerful
- Learn what interactive tests are
- Understand the importance of testing
- Write doctests using documentation
- Properly document Python modules and functions
- Identify and test edge cases

## Requirements

- Ubuntu 20.04 LTS
- Python 3.8.5
- Allowed editors: vi, vim, emacs
- All files must end with a new line
- First line of each file must be `#!/usr/bin/python3`
- Code must follow pycodestyle (v2.7.*)
- All files must be executable
- A README.md file is required

## Testing

- Doctests are located in the `tests` directory
- Run doctests with:
  ```bash
  python3 -m doctest ./tests/*


## Run unittests with:

python3 -m unittest tests.6-max_integer_test
