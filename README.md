# python-hello_world

This directory contains introductory Python scripts.
The tasks focus on basic Python concepts such as printing,
variables, strings, formatting, and simple string manipulation.

## Files

- `0-print.py` — Print a simple sentence.
- `1-print_integer.py` — Print an integer using f-strings.
- `2-print_float.py` — Print a float with two decimal places.
- `3-print_string.py` — Print a string multiple times.
- `4-play_with_strings.py` — Manipulate and display parts of a string.
- `5-copy_cut_paste.py` — Copy, cut, and paste parts of a string.
- `6-concat.py` — Create and print a new sentence.
- `9-easter_egg.py` — Print The Zen of Python.

## Requirements
- Python 3
- All scripts must start with `#!/usr/bin/python3`
- Files must be executable
- Code must follow pycodestyle (PEP8)
